Hyperlinks

A hyperlink is a connection from one webpage to another webpage, file, email address, or location within the same page. Hyperlinks allow users to navigate through websites easily by clicking text or images.

Hyperlinks are created using the anchor tag <a>. The destination of the link is placed inside the href attribute.

Basic Syntax

<a href="https://example.com">Visit Website</a>

HTML Links

HTML links are clickable elements that direct users to another page, another section of the same page, a file download, or an email address. Links are one of the most important parts of web navigation.

Types of HTML Links

  • External Link – Opens another website.
  • Internal Link – Links to another page in the same website.
  • Anchor Link – Links to a section within the same page.
  • Email Link – Opens the email application.
  • Image Link – Uses an image as the clickable link.

Example of Email Link

<a href="mailto:example@email.com">Send Email</a>

Link Attributes

Attributes provide additional information about the hyperlink. They modify how the link behaves when clicked.

  • href – Specifies the destination URL.
  • target – Defines where the link will open.
    • _self – Opens in the same tab
    • _blank – Opens in a new tab
    • _parent – Opens in the parent frame
    • _top – Opens in the full window
  • title – Shows extra information when hovering over the link.
  • rel – Defines the relationship between the current page and the linked document.

HTML Input

HTML inputs are elements used inside forms to collect information from users. They allow users to type text, select options, upload files, and perform other actions.

Input elements are created using the <input> tag and are commonly placed inside a <form> element.

Basic Syntax

<input type="text">

Types of HTML Inputs

  • text – Single line text input
  • password – Hidden text for passwords
  • email – Email address input
  • number – Numeric input
  • checkbox – Multiple selection options
  • radio – Select one option
  • file – Upload files
  • submit – Submit the form
  • reset – Reset the form fields
  • date – Select a date

Input Attributes

Attributes control how input fields behave and how data is collected.

  • type – Defines the type of input.
  • name – Identifies the input when submitting form data.
  • value – Specifies the default value.
  • placeholder – Displays hint text inside the input field.
  • required – Makes the field mandatory.
  • maxlength – Limits the number of characters.
  • readonly – Prevents editing.
  • disabled – Disables the input field.
