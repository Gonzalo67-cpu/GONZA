Lesson 1: What is CSS?

CSS (Cascading Style Sheets) describes how HTML elements are displayed.

It separates content (HTML) from presentation (CSS).

Three layers of a webpage:

  • Content layer – HTML
  • Presentation layer – CSS
  • Behavior layer – JavaScript

Lesson 2: Types of Style Sheets

There are 3 types of CSS:

  1. Inline – inside the tag using style=""
  2. Internal – inside the <style> tag
  3. External – linked with a separate .css file

Example of a CSS Rule:

selector { property: value; }

Lesson 3: Display, Div, and Span

<div> is a block-level container. It starts on a new line.

<span> is inline and used to style text within a line.

display: block; display: inline; display: none;

Lesson 4: CSS Selectors

Common Types of Selectors

  • Universal Selector: * { margin: 0; }
  • Element Selector: p { color: blue; }
  • Class Selector: .note { background: yellow; }
  • ID Selector: #top { background: navy; }
  • Group Selector: h1, h2, p { text-align: center; }
  • Descendant Selector: div p { color: green; }
  • Child Selector: ul > li { font-weight: bold; }
  • Pseudo-class Selector: a:hover { color: red; }
