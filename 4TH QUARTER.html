Lesson 1: Events & Pop-up Boxes

Events are actions triggered by user interaction such as mouse clicks, keyboard presses, loading of images, or submitting a form.

Event Handlers help call functions at the correct time so the program responds properly to page events.

Common Event Triggers

  • onclick – mouse click
  • ondblclick – double click
  • onfocus – element gets focus
  • onblur – element loses focus
  • onmouseover – mouse moves over element
  • onmouseout – mouse leaves element
  • onkeydown – keyboard key pressed
  • onkeyup – keyboard key released
  • onsubmit – submit button clicked
  • onload – page finishes loading
  • onunload – user leaves page

Pop-up Boxes

Pop-up boxes display messages or request information from the user.

Alert Box

alert("WARNING!");

Prompt Box

var name = prompt("Please enter your name:");

Confirm Box

var answer = confirm("Do you want to proceed?");

Lesson 2: Conditions, Controls, Loops

Conditional statements allow programs to perform different actions depending on a condition.

If Statement

if(condition){
   //code to execute
}

If-Else Statement

if(grade >= 75){
document.write("Passed");
}
else{
document.write("Failed");
}

If-Else-If Statement

if(answer=="Web Scripting"){
document.write("Correct!");
}
else if(answer==""){
document.write("No answer");
}
else{
document.write("Try again");
}

Switch Statement

switch(book){
case "1":
alert("J.R.R Tolkien");
break;

case "2":
alert("C.S Lewis");
break;

default:
alert("Unknown");
}

Lesson 3: Functions

A function is a reusable block of code that performs a specific task. Functions help avoid repeating code.

Function Syntax

function functionName(var1,var2){
   //code here
}

Example

function add(a,b){
return a+b;
}

The return statement sends the result of a function back to the program.

Lesson 4: Loops

Loops repeat a block of code until a condition is met.

For Loop

for(var i=1;i<=5;i++){
document.write(i);
}

While Loop

while(condition){
   //code here
}

Do-While Loop

do{
   //code here
}
while(condition);

For-In Loop

for(variable in object){
   //code here
}

Break stops a loop while Continue skips to the next iteration.

Lesson 5: String, Array, Date Objects

String Object

A string object represents text and has properties and methods for manipulating characters.

var text = "This is a new house";
document.write(text.length);

Common String Methods

  • charAt()
  • concat()
  • indexOf()
  • replace()
  • search()
  • slice()
  • split()
  • toUpperCase()
  • toLowerCase()

Array Object

An array stores multiple values in a single variable.

var fruits = ["Apple","Banana","Orange"];

Common Array Methods

  • push()
  • pop()
  • concat()
  • slice()
  • sort()
  • join()

Date Object

The Date object is used to manipulate dates and time.

var today = new Date();

Date Methods

  • getDate()
  • getMonth()
  • getFullYear()
  • getHours()
  • getMinutes()

Lesson 6: Timing and Math Object

setTimeout()

Executes code after a specified time.

setTimeout("alert('Hello')",3000);

clearTimeout()

Cancels a timer created by setTimeout().

clearTimeout(timer);

Math Object

Properties

  • Math.E
  • Math.SQRT2

Methods

  • Math.abs(x)
  • Math.ceil(x)
  • Math.floor(x)
  • Math.max(x,y)
  • Math.min(x,y)
  • Math.pow(x,y)
  • Math.random()
  • Math.round(x)
  • Math.sqrt(x)
