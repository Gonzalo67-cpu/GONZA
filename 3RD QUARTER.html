Lesson 1: Introduction to JavaScript

Script – a set of computer instructions or commands that puts together or connects existing components to accomplish a new related task.

JavaScript is a powerful client-side scripting language interpreted by the web browser. It was created by Brendan Eich. JavaScript enables web developers and designers to build more functional and interactive websites.

It was originally called Mocha, later renamed to LiveScript, and finally became JavaScript. It is not associated with the Java programming language. JavaScript is case-sensitive and is usually embedded directly in HTML files.

Characteristics of JavaScript

  • Interpreted Language – requires a browser to run.
  • Scripted Language – used for repetitive tasks.
  • Client-Side Scripting – executed on the user's computer.
  • Object-Oriented Programming (OOP) – treats elements as objects.

JavaScript Terms

  • Objects – items that exist in the browser.
  • Methods – actions performed by an object.
  • Properties – characteristics of an object.
  • Event – something that happens such as clicking or loading a page.
  • Parameter – data describing an object.
  • Type – type of variable such as integer, string, or Boolean.
  • Type Conversion – converting a variable from one type to another.

Lesson 2: More About JavaScript

JavaScript is a client-side, high-level scripting, interpreted and object-oriented language.

  • Client-side – processed on the user's computer.
  • High-level scripting – written using words similar to English.
  • Interpreted – converted into machine code while running.

Things JavaScript Can Do

  • React to events such as mouse clicks.
  • Validate form data.
  • Create cookies to store user information.
  • Enhance webpages with animations and dynamic content.

Features of JavaScript

  • Supports structured programming syntax similar to C.
  • Dynamic typing.
  • Runs locally on a website.
  • Detects user actions like mouse clicks.
  • Can be combined with CSS to create Dynamic HTML.

Disadvantages

  • Some browsers may not support JavaScript.
  • JavaScript code can be viewed by users.
  • Potential security risks from malicious scripts.

Lesson 3: JavaScript Syntax

W3C (World Wide Web Consortium) – organization responsible for standardizing web technologies.

DOM (Document Object Model) – allows access and manipulation of webpage content.

Parsing – analyzing program syntax to extract information.

Syntax Guides

  • Quotes – used to display text exactly as written.
  • Single quotes: ' '
  • Double quotes: " "
  • Comments
  • // single-line comment
  • /* multiple-line comment */
  • Plus (+) – used to concatenate strings.

Lesson 4: Variables and Operators

Variables

Variables store information that can be used later in a program. In JavaScript, variables act as containers that hold values which can change.

Data Types

  • Character
  • String
  • Integer
  • Float / Double
  • Boolean
  • Function
  • Object
  • Array
  • Undefined
  • Null

Declaring Variables

var firstname;
var firstname, lastname;
var firstname = "Ivan";
var firstname = "Kim", lastname = "Sunoo";

Variables can also be declared without using the var keyword.

year = 2020;

Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus
  • ++ Increment
  • -- Decrement

Comparison Operators

  • == Equal to
  • === Identical
  • != Not equal
  • !== Not identical
  • > Greater than
  • >= Greater or equal
  • < Less than
  • <= Less or equal

Lesson 5: JavaScript Operations and Functions

Concatenation – combining two strings into one.

StringConcatenated = String1 + String2;

Example: "peanut" + "butter" = "peanutbutter"

Functions

A function is a block of code that runs only when called. Functions allow code reusability and help perform repetitive tasks.

function popup() {
alert("Hello!");
}

Returning a Value

Functions can return values after performing operations.

function square(x){
return x*x;
}

Lesson 6: Getting Started with JavaScript

The <script> tag is used to include JavaScript in an HTML document.

Commonly Used Objects

  • window – main object of the browser.
  • document – represents the webpage content.
  • math – performs mathematical operations.
  • navigator – identifies browser information.

Events

An event occurs when a user performs an action such as clicking or loading a page.

Where Scripts Can Be Placed

  • Body
  • Head and Body
  • External JavaScript File

JavaScript Guidelines
